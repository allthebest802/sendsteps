Privacy Policy

Last updated: March 2026

🔒

The short version

Helpset is built to be simple and low-stress. Most tools run fully in your browser. We don't ask you to create accounts, and we don't sell or share personal data.

💾

What gets saved

Some tools may save what you type on your device only — for example in your browser's local storage — so it's still there when you come back. Nothing is uploaded to any server unless a tool clearly says it is.

🚫

No ads. No tracking.

We don't use advertising. If we ever add analytics, it will be minimal and privacy-friendly — and we'll update this page to say so clearly.

✉️

Contact

If you have any questions about privacy on Helpset, you can get in touch via the contact form on the homepage.
