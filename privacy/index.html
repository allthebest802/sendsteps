Privacy Policy

SendSteps is built to be simple and low-stress. Most tools run fully in your browser. We don’t ask you to create accounts, and we don’t sell or share personal data.

Some tools may save what you type on your device only (for example in your browser storage) so it’s still there when you come back. Nothing is uploaded unless a tool clearly says it is.

We don’t use ads. If we ever add analytics, it will be minimal and privacy-friendly.
